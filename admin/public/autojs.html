Auto.js 收款自动充值脚本

此脚本用于监听支付宝/微信收款通知，自动调用后端 API 完成充值。

配置你的服务器信息

脚本代码

长按下方代码区域，选择"全选"然后"复制"，粘贴到 Auto.js 中。

使用步骤

1 配置服务器地址

在上方输入你的服务器地址（手机能访问的地址）

2 生成并复制脚本

点击"生成脚本代码"，然后复制生成的代码

3 在 Auto.js 中创建脚本

打开 Auto.js → 点击"+" → 选择"文件" → 粘贴代码 → 保存

4 开启必要权限

设置 → 辅助功能 → Auto.js → 开启无障碍服务

设置 → 通知 → Auto.js → 允许读取通知

5 运行脚本

在 Auto.js 中点击运行按钮，脚本将开始监听收款通知

注意：
  • 手机需要保持屏幕常亮或在后台运行
  • 支付宝/微信需要开启收款通知
  • 脚本只监听通知栏的收款消息，不会读取聊天内容
